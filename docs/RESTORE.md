# استرجاع نسخة احتياطية (Restore)

إجراءات لاسترجاع نسخة احتياطية محليًا:

1. انتقل إلى مجلد `backups/` وستجد ملفات بصيغة `.db.gz`.
2. فك الضغط:
   gzip -d backups/amstore-2026-01-11T12-00-00-000Z.db.gz
   سينتج ملف `.db`.
3. (اختياري) احتفظ بنسخة من القاعدة الحالية:
   cp amstore.db amstore.db.before-restore
4. استبدل قاعدة البيانات بالتالي:
   mv backups/amstore-2026-01-11T12-00-00-000Z.db amstore.db
5. أعد تشغيل السيرفر:
   npm run restart  # أو الطريقة التي تستخدمها (pm2, systemd, إلخ)

إذا رفعت النسخة إلى S3، حمّل الملف أولًا ثم فك ضغطه كما سبق.
